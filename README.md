## Instalação e uso

Clone o repositório

```bash
git clone https://github.com/deniscoelho-js/cypress-rest-api-sbr.git

```

Siga os passos abaixo

```bash
# Instale as dependências
$ npm i

# Inicie o servidor
$ npx cypress open
```
